# CS-499 Computer Science Capstone ePortfolio - Justin Harrison

This repository contains the source files for my CS-499 Computer Science Capstone ePortfolio at Southern New Hampshire University.

The ePortfolio showcases my professional self-assessment, code review video, original and enhanced capstone artifacts, and enhancement narratives across the following focus areas:
- Software Engineering and Design
- Algorithms and Data Structures
- Databases

## Live ePortfolio Website
The complete, polished ePortfolio is hosted using GitHub Pages and can be viewed here:

https://justinharrison99.github.io/

## Repository Purpose
This repository exists solely to host the files required for the GitHub Pages site.  
All portfolio content is intended to be viewed through the live website rather than directly through the repository file structure.

## Contents (For Reference Only)
- `index.html` — Main ePortfolio website
- `artifacts/` — Original artifact and enhanced capstone projects
- `code review/` - Access to the full code review video
- `narratives/` — Enhancement narratives for each artifact

For the best experience and full context, view the live ePortfolio using the link above.
